# 🌐 UIKit Networking Example
A UIKit app demonstrating how to fetch, decode, and display data from a REST API using URLSession.

---

## 🤔 What this is
This project shows how to make network requests with URLSession, decode JSON responses with Codable, and update a UIKit interface with the results. It covers async/await networking, error handling, and loading states — the full lifecycle of a real network call in a UIKit app.

## ✅ Why you'd use it
- **URLSession with async/await** — Shows the modern Swift concurrency approach to networking in UIKit
- **Codable JSON decoding** — Demonstrates how to map API responses to Swift model structs cleanly
- **Loading & error states** — Covers how to show activity indicators and handle network failures gracefully in the UI

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/UIKitNetworkingExample.git
cd UIKitNetworkingExample
```

### 2. Open in Xcode
Double-click `UIKitNetworkingExample.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- Requires an active internet connection to fetch live data.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
